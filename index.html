A

Alignement

CSS3 a introduit trois propriétés en lien avec l'alignement d'un élément flexible.

align-content

La propriété align-content est une propriété qui modifie le comportement des lignes d'un élément qui est flex-wrap.

Il existe plusieurs valeurs possible dont :

  • center
  • space-around
  • safe
  • start
  • baseline
  • stretch

La valeur par défaut est :stretch.

							

contenant_exemple { 
		height: 200px;
		width: 250px;
}

.contenant_align_content { 
		align-content: space-around;
		display: flex;
		flex-wrap: wrap;
}

.contenu_align { 
		background: #f95d9b;
		border: 5px solid var(--secondaire);
		height: 50px;
		margin: 5px;
		width: 100px;
}

align-items

La propriété align-items définit la propriété align-self sur l'ensemble des éléments dans un parent flexible.

Il existe plusieurs valeurs possible dont :

  • center,
  • self-end
  • safe
  • start
  • stretch

La valeur par défaut est :stretch.

							

contenant_exemple { 
	height: 200px;
	margin: 10px;
	width: 250px;
}

.contenant_align_content { 
	align-content: space-around;
	display: flex;
	flex-wrap: wrap;
}

.contenant_align_items {
	align-items: space-around;
	display: flex;
	flex-wrap: wrap;
}

align-self

La propriété align-self permet de définir l'alignement d'un élément spécifique qui se retrouve dans un parent flex.

Il existe plusieurs valeurs possible dont :

  • center
  • self-end
  • safe
  • start
  • stretch

La valeur par défaut est :auto.

							

contenant_exemple { 
	height: 200px;
	margin: 10px;
	width: 250px;
}

.contenant_align_content { 
	align-content: space-around;
	display: flex;
	flex-wrap: wrap;
}

.contenant_align_self { 
	display: flex;
	flex-wrap: wrap;
}

.contenant_align_self > .contenu_align:nth-child(2) { 
	align-self: flex-end;
	background: #161748;
	border-color: #f95d9b;
}

B

Les bordures

CSS3 a ajouté des propriétés qui permettent d'avoir des bordures plus interessantes.

border-image

La propriété border-image permet de mettre une image ou un gradient sur la bordure.

C'est une propriété raccourcis qui rassemble les propriétés :

  • border-image-source
  • border-image-width
  • border-image-repeat
  • border-image-slice
  • border-image-outset

La valeur par défaut est :none 100% 1 0 stretch.

							

.contenant_exemple { 
	border: 5px solid var(--principale);
	height: 200px;
	margin: 10px;
	width: 250px;
}

.contenant_border-image_1 {
	border-image: linear-gradient(var(--accent), var(--principale)) 50 / 10;
}

							
						
						
							
.contenant_exemple { 
	border: 5px solid #161748;
	height: 200px;
	width: 250px;
}

.contenant_border-image_2 {
	border-image: url(/images/bordure_pois.jpg) 27 round;
}

border-radius

La propriété border-radius permet de définir à quel point nous souhaitons que les coins soient arrondis.

Lorsque l'on place qu'une seule valeur, nous définissons tous les coins en même temps. Il est possible de définir les coins indépendamment en utilisant une propriété spécifique comme :

  • border-bottom-right-radius
  • border-bottom-left-radius
  • border-top-right-radius
  • border-top-left-radius

Il est aussi possible de définir les coins indépendamment en utilisant seulement la propriété border-radius, pour ce faire, nous devons mettre plusieurs valeurs séparées par un espace.

Une valeur
Tous les coins auront la valeur écrite.
Deux valeurs
La première valeur représentera le coin supérieur gauche et le coin inférieur droit. La deuxième valeur représentera le coin supérieur droit et le coin inférieur gauche.
Trois valeurs
La première valeur représente le coin supérieur gauche. La seconde valeur représente le coin supérieur droit et le coin supérieur gauche. La dernière valeur représente le coin inférieur droit.
Quatre valeurs
La première valeur représente le coin supérieur gauche. La deuxième valeur représente le coin suppérieur droit. La troisième valeur représente le coin inférieur droit. La dernière valeur représente le coin inférieur gauche.

La valeur par défaut est :0.

							

.contenant_exemple { 
	border: 5px solid #161748;
	height: 200px;
	width: 250px;
}

.contenant_border-radius_1 { 
	border-radius: 50%;
}

							
						
							

.contenant_exemple { 
	border: 5px solid #161748;
	height: 200px;
	width: 250px;
}

.contenant_border-radius_2 { 
	border-radius: 50% 10px;
}

							
						
							

.contenant_exemple { 
	border: 5px solid #161748;
	height: 200px;
	width: 250px;
}

.contenant_border-radius_3 { 
	border-radius: 50% 10px 15px;
}

							
						
							

.contenant_exemple { 
	border: 5px solid #161748;
	height: 200px;
	width: 250px;
}

.contenant_border-radius_4 { 
	border-radius: 50% 10px 15px 35%;
}

F

Les filtres

Il est possible de rajouter des effets sur des éléments avec des filtres.

Nous utilisons principalement les filtres sur des images.

filter

La propriété filter permet d'obtenir des effets sur un élément.

Elle regroupe plusieurs valeurs dont :

  • blur
  • brightness
  • contrast
  • drop-shadow
  • invert
  • opacity

La valeur par défaut est :none.

							

.filter_blur { 
	filter: blur(5px);
}

							
						
							

.filter_brightness { 
	filter: brightness(50%);
}								
							
						
							

.filter_contrast { 
	filter: contrast(50%);
}

							
						
								

.filter_drop-shadow { 
	filter: drop-shadow( 0 10px 10px var(--noir));
}	

							
						
							

.filter_grayscale { 
	filter: grayscale(85%);
}

							
						
							

.filter_hue-rotate { 
	filter: hue-rotate(300deg);
}								
							
						
							

.filter_invert { 
	filter: invert(100%);
}	

							
						
							

.filter_opacity { 
	filter: opacity(50%);
} 

							
						
							

.filter_saturation { 
	filter: saturate(400%);
} 
								
							
						
							

.filter_sepia { 
	filter: sepia(50%);
} 

							
						
							

.filter_multiples { 
	filter: contrast(50%) 
	brightness(120%) 
	saturate(500%)
	invert(80%);
}

Overflow

Lorsqu'un élément est plus grand que son conteneur, il peut être intéressant de pouvoir déterminer le comportement de cet élément.

overflow

La propriété overflow détermine comment afficher un élément qui est plus grand que son conteneur.

Les valeurs possibles sont :

  • visible
  • hidden
  • scroll
  • auto

La valeur par défaut est : visible.

							

.contenant_exemple {

	width: 250px;
}

.img_renard { 
	width: 300px;

	width: 250px;

.img_renard { 
	width: 300px;
}

.overflow_hidden { 
	overflow: hidden;
}

							
						
							

.contenant_exemple {
	height: 200px;
	width: 250px;
}

.img_renard { 
	width: 300px;
}

.overflow_scroll { 
	overflow: scroll;

Perseptive

Il est possible de créer des éléments 3D avec dy CSS.

perseptive

La propriété perseptive permet d'établir la longeur de l'axe Z.

La valeur par défaut est : none.

							

.exemple_perseptive { 
	height: 200px;
	overflow: hidden;
	perspective: 100px;
